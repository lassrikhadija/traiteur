# Maison Safran — démo traiteur

Site vitrine de **démonstration** pour un traiteur événementiel haut de gamme à Montréal.

- **Marque, plats, prix, avis et coordonnées : 100 % fictifs** (illustration uniquement).
- Cuisine **100 % halal, sans alcool** — aucune mention de vin ni de porc.
- Page autonome : un seul `index.html` (CSS + JS inline) et un dossier `img/` de photos.
- Photos d'illustration libres de droits (Pexels).

Conçu par **[nextiweb.ca](https://nextiweb.ca)** — agence web (création de sites, SEO, marketing digital), Montréal.

## Aperçu local

Ouvrir `index.html` dans un navigateur, ou servir le dossier :

```bash
php -S 127.0.0.1:3456
# puis http://127.0.0.1:3456/
```
